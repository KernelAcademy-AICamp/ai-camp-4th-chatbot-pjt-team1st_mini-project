# 🏛️ 국립중앙박물관 국보 퀴즈 챗봇

국립중앙박물관 소장 국보 15점에 대한 인터랙티브 퀴즈 챗봇입니다.

## ✨ 주요 기능

- **유물 선택**: 15개 국보 중 3~10개 선택하여 퀴즈 진행
- **동적 퀴즈 생성**: Google Gemini API를 활용한 AI 퀴즈 자동 생성
- **맞춤형 해설**: 정답/오답에 따른 AI 맞춤 해설 제공
- **공식 이미지**: 국립중앙박물관 공식 유물 이미지 표시

## 🛠️ 기술 스택

- **Frontend**: Streamlit
- **LLM**: Google Gemini API (gemini-2.0-flash)
- **Language**: Python 3.10+

## 🚀 실행 방법

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 환경 변수 설정
`.env` 파일 생성:
```
GEMINI_API_KEY=your_api_key_here
```

### 3. 실행
```bash
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 접속

## 📁 프로젝트 구조

```
museum-chatbot/
│
├── app.py                    # 메인 Streamlit 앱
│
├── config/
│   ├── styles.py             # CSS/색상/폰트 스타일
│   ├── prompts.py            # AI 프롬프트 템플릿
│   └── settings.py           # 앱 설정
│
├── data/
│   └── artifacts.py          # 국보 15개 데이터 (이미지 URL 포함)
│
├── services/
│   ├── llm_service.py        # Gemini API 연동
│   └── museum_api.py         # 박물관 API 연동 (선택)
│
├── .env.example              # 환경변수 예시
├── requirements.txt          # 패키지 목록
└── README.md
```

## 👥 역할별 담당 파일

| 역할 | 담당 파일 | 수정 내용 |
|------|----------|----------|
| **디자이너** | `config/styles.py` | 색상, 폰트, CSS, 레이아웃 |
| **기획자/PM** | `config/prompts.py` | AI 말투, 메시지 |
| **콘텐츠** | `data/artifacts.py` | 유물 정보, 퀴즈 추가/수정 |
| **개발자** | `app.py`, `services/` | 기능 로직 |
